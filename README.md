# 🚀 PromptEra

> **The Era of Intelligent Conversations.**

PromptEra is a modern AI-powered chatbot built using **Flask**, **JavaScript**, and **Google Gemini API**. It provides a clean conversational interface where users can interact with an AI assistant in real time.

🌐 **Live Demo:** https://promptera-553s.onrender.com

---

## ✨ Features

- 🤖 AI-powered chatbot
- 💬 Real-time conversations
- 📂 Chat history
- 🔍 Search previous chats
- 🗑 Delete individual chats
- 🆕 Create new conversations
- 📱 Responsive UI
- ⚡ Fast Flask backend
- 🔐 Environment variable support for API keys

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/tanishqkamboj5-rgb/PromptEra.git
```

Move into the project

```bash
cd PromptEra
```

Install dependencies

```bash
pip install -r backend/requirements.txt
```

Create a `.env` file inside the backend folder

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run the application

```bash
cd backend
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
